# Prompt Collection

This repository contains prompts that you can use with ChatGPT for web design extraction and with Google AI Studio for portfolio creation workflows.

## Features

- Extract web design structure and styling ideas
- Generate portfolio website concepts
- Reuse prompts across different AI tools
- Simple and customizable prompt setup

## Supported Platforms

- ChatGPT
- Google AI Studio

## Use Cases

- Web Design Extractor
- Portfolio Creator
- UI Inspiration
- Landing Page Generation

## How to Use

1. Copy the prompt you want to use
2. Paste it into ChatGPT or Google AI Studio
3. Customize the input based on your project
4. Generate results instantly

## Notes

These prompts are designed to help speed up web design and portfolio development workflows using AI.
